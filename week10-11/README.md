# JS Videoclipper

![Image](https://preview.ibb.co/d6TxgK/2018_08_06_02_28_12.png)

## Requeriments: 

*Node.js (NPM) 

## Installation: 

You can install it running the following commands: 

```$ yarn install```

or 

```$ npm install ```

Also there is a built version wihout editing capabilities in `/build` folder

You can run this build wihout installing `npm` or any dependency just putting the contents of that folder into a WebServer.

`index.html` is a container of the app, it was made to check that is possible to embed the app in any webpage using `<iframes>` to  inject `videoClipper.htm` what is the main application. 

if you access `videoClipper.html` directly you will get the app running in all the browser. 

### Usage tips:
Hotkey controls are `alt+b` for previous video and `alt+n` for next video
